# PyCharm Community 版 - 下载与安装说明

## 1. 下载状态

- **安装包**：PyCharm Community 2025.2.6（Windows .exe）
- **保存位置**：`C:\Users\Laptop\Downloads\pycharm-community-2025.2.6.exe`
- **大小**：约 768 MB

下载已在后台进行。若该文件尚未达到约 768 MB，请稍等下载完成。

若下载中断或未开始，可手动下载：
- 官方直链：https://download.jetbrains.com/python/pycharm-community-2025.2.6.exe
- 或打开：https://www.jetbrains.com/pycharm/download/?section=windows 选择 Windows 版本下载。

---

## 2. 安装步骤

1. **运行安装程序**  
   在资源管理器中双击：
   ```
   C:\Users\Laptop\Downloads\pycharm-community-2025.2.6.exe
   ```
   或在 PowerShell 中执行：
   ```powershell
   Start-Process "$env:USERPROFILE\Downloads\pycharm-community-2025.2.6.exe"
   ```

2. **安装向导**
   - 若提示 UAC，选择“是”。
   - 选择安装路径（默认即可）。
   - 按需勾选：
     - 创建桌面快捷方式
     - 将 PyCharm 加入 PATH（可选，便于命令行启动）
   - 点击 **Install**，等待完成。

3. **完成**
   - 安装结束后可选择“Run PyCharm Community Edition”立即启动。
   - 首次启动可能要求选择主题、插件等，按提示操作即可。
